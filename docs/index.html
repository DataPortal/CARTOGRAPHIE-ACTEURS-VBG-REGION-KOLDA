Contenu & cohérence des données

La base est structurée pour permettre : (i) annuaire opérationnel, (ii) cartographie des services, (iii) analyses gaps, (iv) coordination/référencement.

  • Champs clés : structure, type, services, cibles, zone, contacts, lat/lon.
  • Typologie VBG : prévention, médical, psychosocial, juridique, autonomisation.
  • Bonnes pratiques IM : standardisation + mise à jour contrôlée.
